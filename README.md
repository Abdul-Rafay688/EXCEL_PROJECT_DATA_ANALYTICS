## Introduction

This repository contains my Excel projects built while following Luke Barousse's **Excel for Data Analytics** course. Using a real-world dataset of data job postings, I built an interactive salary dashboard and a deeper skill/salary analysis workbook to practice turning raw data into insights.

### 📁 Projects

## - 📊 [Salary Calculator Dashboard]
  An interactive dashboard where you pick a Job Title, Country, and Type from dropdowns and instantly see the matching median salary and charts.  
  [Checkout my work here](Project_1_Dashboard)

## - 🔎 [Skill & Salary Analysis]
  A Power Query + PivotTable driven analysis answering which skills are most in demand and which pay the most.  
  [Checkout my work here](Project_2_Analysis)

### 🛠️ Excel Skills Used

Across both projects:

- **📉 Charts** — bar charts, formatted for clarity
- **🧮 Formulas and Functions** — dynamic arrays, array formulas, `XLOOKUP`, `COUNTIFS`
- **❎ Data Validation** — dropdown-driven filtering
- **🔗 Power Query** — importing and shaping raw data
- **📊 PivotTables** — summarizing and comparing salary/skill data

### 📦 Dataset

Both projects use the same real-world data job postings dataset from the course, containing:

- **👨‍💼 Job titles**
- **💰 Salaries**
- **📍 Locations**
- **🛠️ Skills**

Building these two projects helped me go from raw job posting data to dashboards and insights a recruiter or job seeker could actually use. Check out each project folder below for the full breakdown.
